# agents-sandbox

My agents sandbox.

## Setup

Install the dependencies.

```bash
make init
```

## Development

Run the tests.

```bash
make tests
```

## Production

Build the library.

```bash
make build
```

## Miscellaneous

### Format / Lint

Format the codebase.

```bash
make write
```

Lint the codebase.

```bash
make check
```

### Cleanup

Clean up the codebase.

```bash
make clean
```
